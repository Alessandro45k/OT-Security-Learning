# OT-Security-Learning 

## Overview
This repository documents my journey and technical progress in **Industrial Cybersecurity (OT)**. My goal is to specialize in protecting critical infrastructure and SCADA systems.

## Tools Included

Industrial Fuzzer (Python)
A specialized automation tool designed to discover sensitive configuration files in industrial PLC web interfaces.

* **Functionality:** Analyzes HTTP status codes to identify hidden directories.
* **Key Feature:** Automated exfiltration of discovered files (`.bin`, `.sql`, `.key`) into a local loot folder.
* **Goal:** Efficiency in large-scale industrial audits where manual discovery is not feasible.

## Skills Demonstrated
* **Python Scripting:** Network interaction using the `requests` library.
* **Networking:** Understanding of the TCP/IP stack and industrial ports (102, 502).
* **Automation:** Reducing manual audit times through code.
